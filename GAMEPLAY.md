# Jump Jump Game - 游戏玩法详细指南

## 🎯 游戏目标

在这个跳跃平台游戏中，你的目标是控制角色不断向上跳跃，尽可能获得更高的分数。游戏采用物理引擎驱动，需要精确的时机把握和策略规划。

## 🕹️ 基础操作

### 控制方式
- **鼠标左键**：按住蓄力，松开跳跃
- **R键**：游戏结束后重新开始

### 蓄力跳跃机制
1. **按住** 鼠标左键开始蓄力
2. **蓄力条** 会逐渐增长（最大值：15.0）
3. **松开** 释放跳跃，蓄力越高跳得越远
4. 角色会以 60° 角度向前跳跃
5. 跳跃方向会根据角色朝向自动调整

### 物理系统
- **重力加速度**：0.4（向下拉力）
- **墙壁反弹**：碰到屏幕边缘会自动反弹并改变方向
- **着陆检测**：必须从上方落到平台上才能成功着陆

## 🏗️ 平台系统

游戏中共有 **5种不同类型** 的平台，每种都有独特的颜色和分数奖励：

### 平台类型详解

| 平台类型 | 颜色 | 分数奖励 | 出现概率 | 特点 |
|---------|------|---------|---------|------|
| **普通平台** | 灰色 | +1分 | 基础平台 | 最常见的平台类型 |
| **音乐盒平台** | 粉色 | +40分 | 稀有 | 最高分数奖励 |
| **便利店平台** | 紫色 | +15分 | 较稀有 | 中等分数奖励 |
| **魔方平台** | 黄色 | +10分 | 一般 | 中低分数奖励 |
| **窨井盖平台** | 棕色 | +20分 | 常见 | 中等分数奖励 |

### 平台生成规律
- **间距**：80-180像素（随难度增加）
- **位置**：水平位置随机生成
- **特殊平台概率**：20%-40%（随难度增加）
- **动态生成**：到达倒数第5个平台时自动生成新平台

## 🎮 评分系统

### 基础评分规则
1. **普通着陆**：+1分
2. **精准着陆**（平台中心25%范围内）：触发连击系统
3. **特殊平台**：根据平台类型获得额外分数

### 连击系统
- **触发条件**：连续精准着陆在平台中心
- **连击倍数**：每次连击分数 = 连击数 × 2
- **连击重置**：非精准着陆时连击数重置为1
- **最大收益**：连击越高，单次得分越多

### 评分示例
```
普通着陆音乐盒平台：1 + 40 = 41分
精准着陆音乐盒平台（连击×3）：(3×2) + 40 = 46分
```

## ⚡ 道具系统

游戏中有 **4种道具**，每种都提供独特的增益效果：

### 道具类型详解

#### 🔴 速度提升 (Speed Boost)
- **颜色**：橙色
- **效果**：跳跃力量增加50%
- **持续时间**：20秒
- **策略**：用于跨越更远距离的平台

#### 🟠 双倍分数 (Double Score)
- **颜色**：橙色
- **效果**：所有得分翻倍
- **持续时间**：15秒
- **策略**：在高分平台区域使用效果最佳

#### 🔵 额外跳跃 (Extra Jump)
- **颜色**：青色
- **效果**：获得一次额外跳跃机会
- **持续时间**：20秒
- **策略**：紧急情况下的救命稻草

#### 🟣 慢动作 (Slow Motion)
- **颜色**：紫色
- **效果**：游戏速度减慢50%
- **持续时间**：8秒
- **策略**：精确控制和瞄准时使用

### 道具生成规律
- **出现概率**：15%（随难度降低至5%）
- **位置**：平台上方30像素处
- **收集**：角色碰触即可获得
- **叠加**：同类型道具效果可以刷新时间

## 📈 难度系统

### 动态难度调整
游戏根据你的得分自动调整难度：

#### 难度等级计算
```
难度等级 = 当前分数 ÷ 100
```

#### 难度影响因素

| 难度等级 | 平台间距 | 特殊平台率 | 道具概率 |
|---------|---------|-----------|---------|
| 0-1级 | 80-180px | 20-25% | 15% |
| 2-3级 | 100-200px | 25-30% | 13% |
| 4-5级 | 120-220px | 30-35% | 11% |
| 6级+ | 130-230px | 35-40% | 5% |

### 游戏节奏变化
- **前期**：间距较小，容易上手
- **中期**：引入更多特殊平台和道具
- **后期**：挑战性大幅增加，需要精确操作

2. **道具收集**：彩色粒子（对应道具颜色）
3. **物理模拟**：粒子有重力和随机运动

### UI显示信息
- **当前分数**：实时显示
- **最高分**：历史最佳成绩
- **蓄力值**：充能时显示当前力量
- **道具状态**：显示激活的道具和剩余时间

### 摄像机系统
- **平滑跟随**：摄像机缓慢跟随玩家移动
- **预测性移动**：稍微超前显示上方平台
- **边界限制**：确保游戏画面稳定

## 🏆 高级策略技巧

### 基础技巧
1. **观察平台布局**：提前规划跳跃路线
2. **掌握蓄力时机**：根据距离调整力量
3. **利用墙壁反弹**：扩大可达范围
4. **保持连击**：优先瞄准平台中心

### 进阶策略
1. **道具组合使用**：
   - 双倍分数 + 特殊平台 = 最大收益
   - 慢动作 + 精准着陆 = 稳定连击
   - 速度提升 + 远距离跳跃 = 探索更多路径

2. **风险评估**：
   - 安全路线 vs 高分路线的选择
   - 道具使用时机的把握
   - 连击保持 vs 分数最大化的平衡

3. **难度适应**：
   - 低等级：专注连击建立
   - 中等级：平衡安全和收益
   - 高等级：保守策略，确保生存

## ⚠️ 游戏结束条件

### 失败条件
- 角色跌落到屏幕底部
- 无法到达任何平台

### 重新开始
- 按 **R键** 立即重新开始
- 最高分会保留（当前会话内）
